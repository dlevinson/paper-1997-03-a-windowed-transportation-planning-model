# Decoded SLATE Model Outputs

This is the primary reader-facing copy of the legacy SLATE model outputs decoded from HP-UX-era `.Z`, `.z`, `.Z-A`, and related files. Encoded originals are preserved once in `../legacy_encoded_originals/legacy_encoded_originals_20260516.zip`.

Some filenames include `__from_*` when two encoded originals decode to the same clean filename.
